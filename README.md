# Divi Chopra — Full Stack Developer Portfolio

A simple, responsive portfolio website showcasing projects, skills, experience, and contact information.

## Live / Preview
- Open the site locally by opening the `frontend/index.html` file in your browser.

## Features
- Clean, responsive hero section with profile image and call-to-action buttons.
- Sections for About, Experience, Projects, Skills, and Contact.
- Simple contact form (front-end only) and links to GitHub / LinkedIn.

## Tech Stack
- HTML, CSS, JavaScript
- Designed for static hosting (GitHub Pages, Netlify, Vercel)

## Project Structure
- `frontend/` — contains the website source files
  - `frontend/index.html` — main HTML page
  - `frontend/styles.css` — stylesheet
  - `frontend/script.js` — JavaScript interactions

## Run Locally
1. Clone or copy the project to your machine.
2. Open the `frontend/index.html` file in any modern browser.

```html
<!-- Example: open file in browser by double-clicking or via terminal -->
start frontend/index.html
```

## Deployment
- Recommended: push repository to GitHub and enable GitHub Pages (serve from `main` / `gh-pages`) or deploy the `frontend/` folder with Netlify / Vercel.
